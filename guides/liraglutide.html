Liraglutide

Daily GLP-1 protocol guide centered on adherence and site rotation.

Dosing and Protocol Context

For Liraglutide, protocol moves are safer when escalation is tied to documented tolerance windows. [C1]

Common protocol patterns

  • Use fixed weekly checkpoints for Liraglutide so appetite and GI trends stay comparable across titration steps. [C2]
  • Avoid escalation decisions based on one difficult day; use full trend windows before protocol changes. [C3]

Monitoring windows

  • Track nausea, satiety, bowel pattern, and hydration in relation to dose day and meal timing. [C2]
  • Document escalation pauses and symptom recovery before any further dose progression. [C3]

What to Track in ShotClock

  • Track appetite return, meal size tolerance, and GI patterns around each Liraglutide dose window.
  • Log dose timing, hydration, and bowel pattern in a consistent format for week-to-week comparison.

Safety and Red Flags

  • Stop use and seek clinical guidance if unexpected adverse events occur.
